# Multi-Agent Search - CS 188 Fall 2024

In this project, I designed agents for the classic version of Pacman, including ghosts. Along the way, I implemented both minimax and expectimax search and try my hand at evaluation function design.

### My task
In this assessment, **my task** is to fully complete Q1-Q5 in the original [UC Pacman Project 2 - Multi-Agent Search](https://inst.eecs.berkeley.edu/~cs188/fa24/projects/proj2/)
